MCP Tools — API Transformation

list_books ← GET /books
get_book ← GET /books/{id}
create_book ← POST /books
update_book ← PUT /books/{id}
delete_book ← DELETE /books/{id}
borrow_book ← POST /borrow
return_book ← POST /return
list_genres ← GET /genres

Quick Start

1
Setup: ./scripts/setup.sh — installs all Python venvs + Node.js deps + TypeScript compile
2
Ollama (optional): ollama pull llama3.2 — enables LLM-powered agent mode
3
Start all: ./scripts/start-all.sh — launches all 4 apps in the background
4
Explore: Open the 4 URLs and try: "Show all Fantasy books" in App 4's chat
5
Stop all: ./scripts/stop-all.sh
